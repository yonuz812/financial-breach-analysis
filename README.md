# Financial Breach Analysis

Supporting code and figures for the paper:  
**"The Cost of Data Breaches in the Financial Sector: A Six-Year Empirical Analysis (2020–2025)"**  
Sibel Yonuz — Constructor University Bremen, 2026

## Contents

- `generate_plots.py` — Python script that generates all four figures used in the paper
- `fig1_cost_trend.pdf` — Breach cost trend: financial sector vs global average (2020–2025)
- `fig2_premium.pdf` — Financial sector cost premium over the global average
- `fig3_root_causes.pdf` — Root causes of data breaches (IBM 2025)
- `fig4_attack_vectors.pdf` — Attack patterns in the financial sector (Verizon DBIR 2024)

## Data sources

- IBM Security, *Cost of a Data Breach Report* (2020–2025): https://www.ibm.com/reports/data-breach  
- Verizon, *2024 Data Breach Investigations Report*: https://www.verizon.com/business/resources/reports/dbir/
